# XYZ Bookstore Page

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic bookstore webpage called **XYZ Bookstore**. It displays a small collection of books, each with a title, short description, and a **Buy Now** button. It also includes buttons for **View Cart** and **Checkout**.

The aim of the project is to practise structuring content on a webpage using HTML.

## Features

- Main page heading and introduction
- Book cards with titles and descriptions
- Buy Now buttons for each book
- View Cart and Checkout buttons
- Clear and simple page structure

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using headings and paragraphs
- Grouping content with `div` elements
- Adding buttons to a webpage
- Structuring repeated content in a simple layout
- Organising a page into sections
